# Horse Racing Game

This project is an interactive horse racing game developed using Vue 3, Tailwind CSS, Element Plus and npm.

Demo Link : <a href="http://horseracing.salihangun.com/">http://horseracing.salihangun.com/</a>

## Getting Started

```bash
pnpm install
pnpm run dev
```

## Technologies

- Vue 3 + Vite
- Tailwind CSS
- Element Plus
- Vuex
- pnpm
